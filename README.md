## 📋 Overview
Sideband is an early-stage TypeScript application designed for messaging and remote procedure calls (RPC). It uses features like binary frames and timestamps to ensure quick and efficient communication. We also focus on security with acknowledgments (ACKs) to confirm message delivery.

Whether you want to connect with friends or explore decentralized technology, sideband provides a straightforward solution. P2P messaging, browser compatibility, and command-line usage are on the horizon.

## 🌟 Key Features
- **Fast Messaging:** Quickly send and receive messages.
- **Decentralized Protocol:** Engage in peer-to-peer communication without a central server.
- **Cross-Platform Support:** Works on different systems (Windows, Mac, and Linux).
- **Memory Transport:** Efficient use of resources ensures smooth operation.
- **Runtime Helpers:** Useful tools to aid in running the application with ease.

## 💻 System Requirements
To run sideband, your device should meet the following requirements:

- **Operating System:** Windows 10 or later, macOS 10.14 or later, or any modern Linux distribution.
- **Memory:** At least 4 GB of RAM.
- **Storage:** Minimum of 100 MB of free space.
- **Internet Connection:** Required for messaging features.
